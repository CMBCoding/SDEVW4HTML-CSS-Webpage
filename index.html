Welcome to World of HTML & CSS!

Introduction

What is HTML?

[HyperText Markup Language. A document made up of tags that provide the overall structure and content of a website]

What is CSS?

[Cascading Style Sheet. A sheet made up of rules that determine things such as color, size, position, spacing, etc. of HTML elements.]

HTML Basics

What is a tag?

[Tags are individual, often paired, blocks of code that make up the basic HTML structure. By nesting sementatic tags and filling them with content, a webpage is built.

What is and element?

[Elements are the sum of the opening and closing tags and the content between them]

What are attributes?

[Attributes are modifiers that can be placed in certain tags to give them addition function/meaning.]

[Examples include:]

  • class=""
  • alt=""
  • controls=""
  • for=""
  • href=""

CSS Basics

What is a selector?

[Selectors target parts of the HTML doc based upon specificity. The more specific, the higher priority the styling has]

What is a class?

[A type of selector that can be reused through the document to apply similar stylings to multiple elements]

What is an ID?

[A type of selector that has higher specificity/priority than a class that can only be assigned to one element]
